# CISCO - Circonscription Scolaire Mahajanga

Plateforme complète de gestion des inscriptions aux examens officiels (CEPE, BEPC, BAC) pour la circonscription scolaire.

## Liens en ligne
Frontend : https://sisco-majunga.vercel.app  
Backend : https://sisco-backend.onrender.com  
Admin : `admin` / `admin123`  
Établissement : `etab_401011087` / `sisco2024`

## Fonctionnalités
- Inscription en ligne aux examens
- Espace établissement avec upload de fichiers
- Tableau de bord admin complet
- Chat en temps réel avec Socket.IO
- Système de réinitialisation de mot de passe
- Recherche et filtre des établissements
